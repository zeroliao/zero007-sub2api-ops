# v009 FlClash 分流规则优化

## 范围

- ops 版本：`release/009`。
- 源码版本：沿用线上应用镜像，本次仅调整 Clash 订阅配置生成逻辑。

## 变更

- 完整 Clash 订阅规则改为更接近主流 Mihomo/FlClash 分流模板：
  - 内网、本地域名和保留 IP 段直连。
  - `GEOSITE,private`、`GEOSITE,cn`、`.cn` 域名直连。
  - `GEOSITE,geolocation-!cn` 走 `PROXY`。
  - `GEOIP,CN` 直连。
  - 兜底 `MATCH` 走 `Final`，默认选择 `PROXY`。
- DNS 增加 `geosite:private`、`geosite:cn` fake-ip 过滤，并在 fallback-filter 中加入 `geolocation-!cn`。
- 完整订阅保留服务器 Shadowsocks 节点和 `PROXY` / `Final` 基础分组，减少规则组复杂度。
