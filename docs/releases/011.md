## 变更

- 修复 `codex-auto-review` 上游模型映射，统一转发到 `gpt-5.5`，避免自动审查请求调用旧模型导致失败。
- 新增 API Key 账号模型预览接口，创建账号时可使用当前输入的 API Key 从 OpenAI、Anthropic、Gemini 上游拉取可用模型。
- 优化账号创建页模型白名单交互：API Key 账号显示“获取当前 Key 可用模型”，OAuth 账号显示“填入推荐模型”。
- 更新 OpenAI OAuth 推荐模型列表，只保留当前建议使用的 GPT/Codex/Image 模型，避免把不可用模型误加入白名单。
- 保持不支持 embeddings 模型，本次不新增 `/v1/embeddings` 能力。
