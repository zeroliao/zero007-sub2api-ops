# 005 发布记录

## 目标

把 Clash Verge 订阅从最小节点配置升级为通用规则配置，导入后可直接按规则模式使用。

## 变更

- 订阅 YAML 增加 `mixed-port`、`mode`、`dns` 等通用运行配置。
- 增加多个策略组：
  - `PROXY`
  - `AI`
  - `Streaming`
  - `Microsoft`
  - `Apple`
  - `Final`
- 规则覆盖：
  - 局域网、保留地址直连。
  - 常见广告域名拦截。
  - AI 服务走 `AI` 策略组。
  - 流媒体服务走 `Streaming` 策略组。
  - Microsoft / Apple 可独立选择直连或代理。
  - 常见国内域名和 `GEOIP,CN` 直连。
  - GitHub、Docker、GHCR 等开发服务走代理。
  - 最终规则走 `Final`。

## 验收

- `validate-candidate` 通过。
- 蓝绿部署成功。
- 当前订阅 URL 返回 `200`。
- 订阅 YAML 包含 `dns`、`proxy-groups`、`rules` 和公网 IP 节点地址。
- `clash-node` 容器保持 `healthy`。
