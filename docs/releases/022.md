## 变更内容

- 代理订阅源弹窗主界面改为订阅源列表，新增和编辑订阅源改为独立弹窗。
- 订阅源编辑弹窗集中展示名称、URL、状态、sidecar、纯净度检测和策略指标配置。
- 页面增加全局 `AbuseIPDB API Key` 配置入口；接口不回显明文 Key，空保存不会覆盖旧值，并支持清除页面保存的 Key。
- 订阅源纯净度检测取值优先级调整为：订阅源自定义引用、页面全局 Key、环境变量 `ABUSEIPDB_API_KEY`、`key.md`。
- OpenAI API Key 导入 CCS 时，`endpoint` 自动规范为 `/v1` 版本化入口，避免 CCS 侧拼接 OpenAI-compatible 路径时缺少版本前缀。

## 变更原因

用户希望无需登录服务器即可在页面配置 AbuseIPDB API Key，同时订阅源配置交互需要从单个混合表单优化为“列表 + 新增/编辑弹窗”。此外，OpenAI API Key 导入 CCS 时，OpenAI-compatible 客户端通常需要以 `/v1` 作为 base URL，原来传根地址容易导致客户端请求路径不准确。
